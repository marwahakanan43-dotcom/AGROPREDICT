📚 Common Diseases
🌾 Yellow Rust (Wheat)
Yellow streaks. Use Propiconazole.
🍃 Leaf Blight (Rice)
Brown lesions. Use copper oxychloride.
🪲 Aphids (Mustard)
Sticky leaves. Use Imidacloprid.
⚪ Powdery Mildew
White powder. Use sulfur fungicide.
🟤 Stem Borer (Rice)
Dead heart. Use Chlorpyrifos.
🪲 Bollworm (Cotton)
Holes in bolls. Use Spinosad.
